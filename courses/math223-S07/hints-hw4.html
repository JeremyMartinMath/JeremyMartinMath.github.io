Problem #50 from 2.3

Once you have done part (b), you will suspect that the answer to (c) is "No, they can't possibly exist," because plugging (x,y)=(0,0) into the partial derivatives you found in part (b) will give something undefined.

However, the partial derivatives do exist at (0,0), for a subtle reason. Remember that the partial derivative with respect to x is defined as the derivative of the partial function with respect to x. So you can find the value of the partial derivative fx at (0,0) by calculating

limx → 0 [ f(x,0)-f(0,0) ] / x.

It may help to look at the graph of the surface, either by itself or intersected with the plane y=0.

Problem #28 from 2.5

This problem is really about implicit differentiation in the multivariable setting. Remember that in Math 121, if you have an equation f(x,y)=0, you could find dy/dx implicitly by differentiating both sides with respect to x (treating y not as a constant, but as a function of x) and then solving for dy/dx.

You can do the same thing in this problem: start by differentiating both sides of the equation using the Chain Rule.
